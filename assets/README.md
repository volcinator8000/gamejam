# Assets

Place your art assets here:

```
assets/
  normal/          # Standard office props
    mug.png
    clock.png
    plant.png
    painting.png
    chair.png
    keyboard.png
    ...
  bugged/          # Bug-swapped versions of the same props
    mug_bug.png        # e.g. mug → beetle in a cup
    clock_bug.png      # e.g. clock face → cockroach
    plant_bug.png      # e.g. plant → mantis
    painting_bug.png   # e.g. landscape → spider web
    chair_bug.png      # e.g. chair → ant hill
    keyboard_bug.png   # e.g. keyboard → millipede
    ...
  spider/
    spider_walk.png    # or sprite sheet
  player/
    worker_walk.png
  ui/
    coffee_cup.png
    title_bg.png
```

## Art style tip
For a jam, consider a **pixel art** style (16x16 or 32x32 tiles).
The contrast between clean office pixel art and grotesque bug swaps
will land harder than a realistic style.
