# Input Map (set in Godot Project Settings > Input Map)

| Action     | Default Key       | Purpose                        |
|------------|-------------------|--------------------------------|
| `ui_up`    | W / Arrow Up      | Move up                        |
| `ui_down`  | S / Arrow Down    | Move down                      |
| `ui_left`  | A / Arrow Left    | Move left                      |
| `ui_right` | D / Arrow Right   | Move right                     |
| `interact` | E                 | Examine asset / use machine    |
| `hide`     | F / Space         | Enter/exit hiding spot         |
